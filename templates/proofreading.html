📝 How to Use the Proofreading Tool

1
Select Mode: Click 🖌️ Paint to add mask areas or 🧽 Erase to remove them
2
Adjust Brush: Use the brush size sliders to set the paint/erase brush size
3
Edit Mask: Hold and drag the mouse to paint or erase mask areas. The red overlay shows the current mask
4
Navigate: Use A/D or ←/→ to move between different incorrect layers
5
Save Changes: Click Save to save your edits to the mask file
6
Mark Complete: Click Mark as Corrected when done with this layer

⌨️ Keyboard Shortcuts

A or ← Previous layer
D or → Next layer
P Paint mode
E Erase mode
⌘/Ctrl + Z Undo
⌘/Ctrl + Shift + Z Redo

Mouse: Hold to paint/erase • Zoom: ⌘ Command + Scroll
